# AI-Powered-Social-Media-Engagement-Dashboard
Interactive Power BI dashboard for analyzing social media engagement, audience sentiment, platform performance, content effectiveness, and campaign trends using digital marketing analytics.

Project Overview

This project focuses on analyzing social media engagement data across multiple platforms to measure campaign performance and audience interaction. The dashboard provides actionable insights into engagement trends, platform effectiveness, content performance, and audience sentiment.

Objectives

- Monitor social media engagement metrics.
- Analyze audience sentiment from comments.
- Compare platform performance.
- Evaluate content type effectiveness.
- Support campaign optimization through data-driven insights.

Tools Used

- Power BI
- Microsoft Excel
- Power Query
- DAX

Key Features

- KPI Cards for Total Engagement, Average Engagement Rate, Total Posts, Positive Comments, and Negative Comments.
- Engagement Trend Analysis using Line Charts.
- Platform Performance Comparison.
- Audience Sentiment Distribution.
- Content Type Performance Analysis.
- Interactive Slicers for Platform, Content Type, and Date.

Dashboard Insights

- Tracks likes, comments, shares, and impressions.
- Measures engagement rate across platforms.
- Identifies top-performing content types.
- Analyzes audience sentiment trends.
- Highlights top-performing posts and campaigns.

Business Impact

The dashboard enables marketers to evaluate campaign effectiveness, understand audience reactions, and make data-driven decisions to improve future marketing strategies.
